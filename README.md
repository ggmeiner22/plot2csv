# Plot2CSV

Plot2CSV is a deep-learning pipeline that converts 2D plot images into numerical CSV values.  
It uses synthetic data generation, a ResNet-18 regression model, and automated inference to extract 20 y-axis values from any plot image.

## How to Run

- Run notebook generate_data.ipynb

Outputs:
- training curves
- validation metrics
- Predicted y-values
- Output CSV row
